## Setup with npm
```js
//install sass (for unix-based users root access -sudo- must be provided)
$ npm install -g sass
```

```js
//start sass compiling
$ sass --no-source-map --watch .
```

Then open *index.html* in browser. I recommend using live server extension in vscode
